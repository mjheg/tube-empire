# Hamster Home - Game Design Spec

## Overview

나만의 햄스터를 키우는 힐링 웹게임. 2D 탑뷰 시점에서 케이지 안 햄스터와 상호작용.
터치 반응 + 꾸미기 + 성장 시스템. Vercel 무료 배포, 서버 없이 운영.

## Game Concept

- **장르**: 힐링 가상 펫 시뮬레이션
- **시점**: 2D 탑뷰 (hammyhome 스타일)
- **플랫폼**: 모바일 웹 우선, PC 호환
- **타겟**: 전 연령, 특히 10~30대
- **핵심 감정**: "내 햄스터 너무 귀여워, 또 보고 싶다"

## Hamster Interaction

### 터치/클릭 반응
- **쓰다듬기** (드래그) → 눈 감고 기분 좋은 표정, 하트 이펙트
- **톡톡 치기** (클릭) → 깜짝 놀라서 점프, 고개 갸우뚱
- **먹이주기** (먹이 버튼 → 클릭) → 양볼에 먹이, 볼 빵빵해짐
- **쳇바퀴** → 햄스터가 올라가서 달리기, 속도 점점 빨라짐
- **들어올리기** (길게 누르기) → 발버둥 치다가 잠잠해짐

### 자동 행동 (방치 시)
- 돌아다니기, 먹이 먹기, 쳇바퀴 돌기, 잠자기
- 상태에 따라 행동 변화 (배고프면 먹이 쪽으로, 졸리면 집으로)

### 감정 시스템
- **행복도** - 쓰다듬기, 놀아주기로 올라감
- **배고픔** - 시간 지나면 배고파짐, 먹이 주면 회복
- **피로도** - 놀면 피곤해지고, 자면 회복

행복도 높을수록 귀여운 행동 더 많이 함 (재롱).

## Decoration System

### 기본 제공
- 물병, 먹이그릇, 쳇바퀴, 잠자는 집

### 구매 가능 아이템
- **바닥재** - 나무칩, 모래, 솜, 잔디
- **가구** - 터널, 시소, 해먹, 미끄럼틀, 계단
- **먹이** - 해바라기씨, 당근, 치즈, 딸기 (종류별 다른 반응)
- **집** - 나무집, 성, 텐트, 버섯집
- **테마** - 케이지 배경 (봄 꽃밭, 크리스마스, 우주, 바다)

### 가격 구조
- 일반 아이템 → 게임 내 코인으로 구매
- 프리미엄 아이템 → 광고 시청 또는 실제 결제
- 시즌 한정 → 특별 이벤트 때만

아이템 배치는 케이지 안에서 자유롭게 드래그.

## Growth System

### 성장 단계
| 단계 | 기간 | 특징 |
|------|------|------|
| 아기 | Day 1-3 | 작고 뒤뚱뒤뚱 |
| 청소년 | Day 4-7 | 쳇바퀴 해금, 활발 |
| 어른 | Day 8+ | 재롱, 모든 행동 해금 |

단계마다 외형 변화 (크기, 털 풍성).

### 코인 획득
- 매일 접속 → 코인 지급
- 쓰다듬기 → 소량 코인
- 미니 미션 → "먹이 3번 주기", "쳇바퀴 1분 돌리기"

### 친밀도 레벨
- 상호작용할수록 레벨 상승
- 레벨 올라가면 새 리액션 해금 (하이파이브, 구르기, 점프)

### 행복도 보상
- MAX 유지 → 하루 보너스 코인
- 낮으면 → 구석에서 웅크림 (접속 유도)

## Monetization

### 광고 정책
- 모든 광고는 유저 선택 (강제 없음)
- 보상형 광고: "광고 보면 프리미엄 먹이 1개 무료"
- 1시간에 최대 3번 제한

### 인앱결제
- 프리미엄 가구 - $0.99~$2.99
- 테마 팩 - $1.99
- 광고 제거 - $2.99
- 일반 아이템은 코인으로 무료 구매 가능

## Technical Architecture

### 기술 스택
- **Next.js** - 프레임워크
- **React** - UI
- **HTML Canvas 2D** - 케이지/햄스터 렌더링
- **LocalStorage** - 세이브 데이터
- **Vercel** - 배포 (무료 티어)

### 햄스터 그래픽
- 스프라이트 시트 방식 - 행동별 프레임 이미지
- 걷기, 먹기, 자기, 놀기, 반응 등 상태별 애니메이션
- AI 이미지 생성으로 스프라이트 제작

### 서버 불필요
- 싱글플레이어 → 서버 없음
- 세이브는 LocalStorage
- 서버 비용 = 0원

### 세이브 시스템
- 10초마다 자동 저장
- 오프라인 시간 계산 → "햄스터가 혼자 잘 지냈어요!" + 상태 변화

## Project Structure

```
earn_money/
  game2/                    ← 햄스터 홈
    src/
      app/                  ← Next.js App Router
      components/           ← React UI 컴포넌트
      game/                 ← 게임 로직 (상태, 감정, 성장)
      canvas/               ← Canvas 렌더링 (햄스터, 케이지, 아이템)
      data/                 ← 아이템/미션 데이터
    public/
      sprites/              ← 햄스터 스프라이트 시트
      items/                ← 아이템 이미지
    package.json
```

## 향후 추가 (V2)
- 햄스터 수집 (다양한 종류/색상, 희귀 햄스터)
- 토킹 기능 (마이크로 말하면 높은 목소리로 따라함)
- 한국어/영어 다국어 지원

## Success Criteria

1. 케이지 안에서 햄스터가 자유롭게 돌아다님
2. 터치/클릭 인터랙션 5종 이상 작동
3. 감정 시스템 (행복도/배고픔/피로도) 작동
4. 케이지 꾸미기 (아이템 배치) 작동
5. 성장 시스템 (아기→청소년→어른) 작동
6. 오프라인 시간 계산 작동
7. 모바일/PC 모두 정상 작동
8. Vercel 무료 배포 완료
