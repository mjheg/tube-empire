# YouTuber Tycoon - Game Design Spec

## Overview

방치형 클리커 게임. 유튜버가 되어 구독자 0명에서 미디어 제국까지 키운다.
Vercel 무료 티어, 서버 없이 운영. 수익은 광고 + 인앱결제.

## Game Concept

- **테마**: 유튜버 성장 시뮬레이션
- **장르**: 방치형 클리커 (Idle/Clicker)
- **플랫폼**: 모바일 웹 우선, PC 호환
- **타겟**: 전 세계 10~30대

### 왜 유튜버 테마인가
- 글로벌 공감대 (유튜버 되고 싶은 판타지)
- SNS 공유 욕구가 자연스러움
- 숫자 성장의 쾌감 (구독자 카운터)
- 밈/현실 패러디 요소 풍부

## Core Game Loop

**클릭 → 조회수 → 구독자 → 업그레이드 → 더 많은 조회수 → 반복**

### 자원
- **조회수** - 메인 화폐. 클릭 + 자동 생산으로 획득
- **구독자** - 일정 조회수마다 자동 증가. 영구 수익 배수 역할

### 3단계 진행
1. **직접 촬영** - 클릭 = 영상 업로드 = 조회수 획득
2. **자동화** - 편집자/매니저 고용 → 자동 영상 업로드
3. **스케일업** - 채널 추가, MCN 설립 → 수익 폭발적 증가

### 오프라인 수익
- 앱을 꺼도 시간당 조회수가 쌓임
- 재접속 시 "당신이 없는 동안 142만 조회수를 달성했습니다!" 표시
- "광고 보면 오프라인 수익 2배" 버튼 (핵심 수익 포인트)

### 성장 마일스톤
| 구독자 | 이벤트 |
|--------|--------|
| 100명 | 첫 수익 활성화 |
| 1,000명 | 실버버튼 |
| 10만명 | 골드버튼, 편집자 고용 해금 |
| 100만명 | 다이아버튼, MCN 설립 |
| 1,000만명 | 미디어 제국 |

## Content & Upgrade System

### 영상 카테고리 (해금 순서)
| 순서 | 카테고리 | 해금 조건 |
|------|----------|-----------|
| 1 | 브이로그 | 시작 카테고리 |
| 2 | 게임 | 구독자 500명 |
| 3 | 먹방 | 구독자 5,000명 |
| 4 | 쇼츠/릴스 | 구독자 2만명 |
| 5 | 음악 | 구독자 10만명 |
| 6 | 교육 | 구독자 50만명 |

카테고리마다 조회수 배율이 다름. 여러 카테고리 동시 운영 가능.

### 업그레이드 트리

**장비** (클릭당 조회수 증가)
- 폰 카메라 → 웹캠 → DSLR → 시네마 카메라

**공간** (시각적 성장, 배경 변화)
- 내 방 → 원룸 스튜디오 → 사무실 → 대형 스튜디오

**팀** (자동 생산 속도 증가)
- 혼자 → 편집자 → 썸네일러 → 매니저 → PD

### 랜덤 이벤트
- "영상이 알고리즘에 탔습니다!" → 10분간 조회수 10배
- "악플러 등장" → 미니게임으로 대응하면 구독자 보너스
- "뒷광고 논란" → 구독자 일시 감소, 사과 영상으로 회복
- "콜라보 제안" → 수락하면 일시적 구독자 폭증

### 트렌딩 시스템
- 접속할 때마다 + 30분마다 "지금 뜨는 주제"가 랜덤 변경 (먹방, 챌린지, 밈 등)
- 트렌딩에 맞는 카테고리 영상을 올리면 조회수 2배 보너스
- 매번 접속할 때 확인하고 싶은 이유 제공

### 프레스티지 (환생) 시스템
- 구독자 1,000만명 달성 시 "새 채널로 환생" 가능
- 모든 수치(조회수, 구독자, 장비, 팀) 리셋
- 대신 **영구 보너스** 획득:
  - 환생 1회: 전체 수익 +50%
  - 환생 2회: 시작 시 웹캠 지급
  - 환생 3회: 시작 시 편집자 지급
  - 환생할수록 성장 가속 → 장기 리텐션 핵심
- 환생 횟수는 프로필에 표시 (자랑 요소)

### 일일 보상
- 매일 접속 시 보상 지급
  - 1일차: 조회수 보너스
  - 3일차: 랜덤 장비
  - 7일차: 대박 보상 (프리미엄 스킨 or 대량 구독자)
- 연속 접속 끊기면 1일차부터 다시 시작
- DAU 유지의 기본 장치

## Monetization

### 광고 정책 원칙
- 모든 광고는 유저가 버튼을 눌러야만 재생 (강제 광고 없음)
- 안 봐도 게임 진행에 문제 없음
- 게임 화면에 배너 광고 없음

### 광고 수익 (메인)
- **보상형 광고** - "광고 보면 오프라인 수익 2배" / "30분간 수익 3배"
  - 최대 1시간에 3번까지 제한
- **배너 광고** - 상점/설정 화면에서만 하단 노출
- 중간 강제 광고 없음
- Google AdSense로 시작

### 인앱결제 (서브)
- **광고 제거** - $2.99 일회성
- **프리미엄 패스** - $4.99/월, 수익 2배 + 특별 콘텐츠 해금
- **부스트 팩** - $0.99~$4.99, 조회수/구독자 즉시 획득

### 수익 예상 (보수적)
| DAU | 예상 월 수익 |
|-----|-------------|
| 1,000명 | 30~50만원 |
| 10,000명 | 300~500만원 |
| 50,000명 | 1,500만원+ |

## UI / Screen Layout

### 메인 게임 화면
```
┌─────────────────────────────┐
│  ★ 구독자 1,234명            │
│  ▶ 조회수 5.2M   $ 320만원   │  ← 상단 스탯바
├─────────────────────────────┤
│  ┌───────────────────────┐  │
│  │   실시간 댓글 피드      │  │  ← 댓글이 계속 올라옴
│  │   "ㅋㅋㅋ 개웃기네"     │  │     성장할수록 속도 빨라짐
│  │   "구독 박았습니다"     │  │     악플도 섞임
│  │   "첫 영상부터 봤는데"  │  │
│  └───────────────────────┘  │
│                             │
│     [ 내 방 / 스튜디오 ]      │  ← 배경 + 장비 실시간 반영
│                             │
│    ╔═══════════════════╗    │
│    ║   📹 영상 올리기!  ║    │  ← 메인 클릭 버튼
│    ╚═══════════════════╝    │
│                             │
│  ■■■■■■■□□□ 1만명까지 63%   │  ← 다음 마일스톤 진행바
├─────────────────────────────┤
│  [장비] [팀] [채널] [상점]    │  ← 하단 메뉴 탭
└─────────────────────────────┘
```

### 핵심 UI 요소
- **실시간 댓글 피드**: 자동 생성 댓글이 계속 올라옴. 구독자 적을 때 느리게, 많을 때 폭포처럼. 실제 유튜브 라이브 채팅 느낌.
- **다음 마일스톤 진행바**: 항상 "다음 목표"가 보여서 "조금만 더" 효과.
- **트렌딩 표시**: 현재 뜨는 주제가 상단에 표시. 접속할 때마다 확인 유도.

### 디자인 스타일
- 플랫 일러스트 (심플하고 귀여운 느낌)
- 밝은 색상 위주
- 모바일 우선 (세로 화면), PC에서도 작동

### 시각적 피드백
- 클릭 시 "+120 조회수" 숫자 튀어오름
- 마일스톤 달성 시 축하 팝업 + 이펙트
- 업그레이드 시 배경/캐릭터 변화

## Technical Architecture

### 기술 스택
- **Next.js** - 프레임워크
- **React** - UI
- **Canvas 2D** - 게임 렌더링
- **LocalStorage** - 세이브 데이터
- **Vercel** - 배포 (무료 티어)

### 서버 불필요
- 싱글플레이어 → 서버 없음
- 세이브는 브라우저 LocalStorage
- 광고/결제만 외부 API 연동
- 서버 비용 = 0원

### 페이지 구조
```
/ (메인)
  ├── 게임 화면 (Canvas)
  ├── 상점 모달
  ├── 설정 모달
  └── 마일스톤 팝업
```

한 페이지 앱. 모달로 화면 전환.

### 세이브 시스템
- 10초마다 자동 저장 (LocalStorage)
- 오프라인 수익 = (마지막 접속 시간 ~ 현재 시간) x 초당 수익
- 향후 클라우드 세이브 추가 가능

## Project Structure

```
earn_money/
  game1/                 ← 유튜버 타이쿤
    src/
      app/               ← Next.js App Router
      components/        ← React 컴포넌트
      game/              ← 게임 로직 (상태, 계산, 이벤트)
      data/              ← 업그레이드/카테고리/댓글 데이터
    public/              ← 이미지, 폰트
    package.json
  game2/                 ← (미래) 드리프트 레이싱
  game3/                 ← (미래) IO 멀티플레이어
```

## Success Criteria

1. MVP 출시: 클릭 → 조회수 → 업그레이드 루프가 작동
2. 오프라인 수익 시스템 작동
3. 광고 연동 (보상형 광고 최소 1개)
4. Vercel 무료 티어로 배포 완료
5. 모바일/PC 모두 정상 작동
